# ゆびとま 🍻👆

**「今日飲みに行かない？」を地域で実現するイベントアプリ**

https://yubitoma.shirubelab.jp

---

## これは何？

ゆびとまは、地域の飲み会・イベントをカジュアルに企画・参加できるWebアプリです。

**名前の由来**: 「指が止まる」→ 思わず参加ボタンを押したくなるイベントとの出会い

### 特徴
- 地域イベント・飲み会の投稿・閲覧
- ワンタップで参加表明
- 近日開催のイベントをタイムライン表示
- LINE連携で通知（予定）

---

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| フレームワーク | Next.js (App Router) |
| スタイリング | Tailwind CSS |
| ホスティング | Vercel |
| ドメイン | yubitoma.shirubelab.jp |

---

## 開発

```bash
pnpm install
pnpm dev
# → http://localhost:3000
```

---

## デプロイ

`main` ブランチにpushすると、Vercel経由で自動デプロイされます。
